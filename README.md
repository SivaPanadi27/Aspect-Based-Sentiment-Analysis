# Aspect-Based Sentiment Analysis System

## Overview
This project focuses on analyzing customer reviews using NLP and Machine Learning techniques to perform Aspect-Based Sentiment Analysis (ABSA).

## Current Progress
 Data Collection and Loading
 
 Data Inspection
 
 Data Cleaning
 
 Text Preprocessing

## Tech Stack
- Python
- Pandas
- NLTK
- Scikit-learn
- Streamlit
- Transformers

## Dataset
Amazon Customer Review Dataset collected from Google Dataset Search.
